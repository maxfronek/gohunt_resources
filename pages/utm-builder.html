How to use this tool

(Note: these instructions are for a computer, not a phone or tablet)

  1. Enter the URL you will be linking to in the text box. Make sure to include any and all prefixes (i.e. https://)
  2. Select the relevant parameters from the dropdown lists. If you select Site Marketing, and additional location parameter must be selected. Additional fields will have optional text inputs, which can be left blank if desired.
  3. When ready, click on the Copy Full URL button to copy the complete URL, or click the Copy Parameters Only button to copy only the UTM parameters (the part starting with ?ref=).
  4. Paste your URL wherever you need and it will be trackable through our analytics platforms.
